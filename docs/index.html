Getting Started

Start building beautiful interfaces with Nebula UI in minutes.

Installation

<!-- Add Nebula UI to your project -->
<link rel="stylesheet" href="path/to/nebula-ui.css">

Or download the CSS file directly:

Download Nebula UI

Basic Usage

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Nebula UI Project</title>
    <link rel="stylesheet" href="path/to/nebula-ui.css">
</head>
<body>
    <div class="container">
        <h1 class="title">Hello, Nebula UI!</h1>
        <p>Start building your interface with Nebula UI components.</p>
        <button class="btn btn-primary">Get Started</button>
    </div>
</body>
</html>

Modal Title

This is the modal content. You can put any HTML content here, including forms, images, or text.

Modals are useful for displaying additional information without navigating away from the current page.

<button class="btn btn-primary" id="open-modal">Open Modal</button>

Accordions

Accordions allow users to expand and collapse content sections.

Nebula UI is a comprehensive CSS framework for building modern, responsive, and accessible web interfaces. It provides a wide range of components, utilities, and styles to help you create beautiful websites quickly.

You can install Nebula UI by downloading the CSS file and including it in your HTML, or by using a package manager like npm or yarn.

npm install nebula-ui

Yes, Nebula UI is fully responsive and works well on all device sizes, from mobile phones to large desktop screens. It uses a mobile-first approach to ensure optimal performance on all devices.

            <pre><code>npm install nebula-ui</code></pre>
        </div>
    </div>

// JavaScript to handle accordion functionality
document.querySelectorAll('.accordion-header').forEach(button => {
    button.addEventListener('click', () => {
        const accordionButton = button.querySelector('.accordion-button');
        const expanded = accordionButton.getAttribute('aria-expanded') === 'true';
        
        // Toggle the current accordion
        accordionButton.setAttribute('aria-expanded', !expanded);
        const content = button.nextElementSibling;
        
        if (expanded) {
            content.classList.remove('active');
        } else {
            content.classList.add('active');
        }
    });
});
